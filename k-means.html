KMeans-2

The KMeans algorithm is an unsupervised clustering algorithm that seeks to partition observations into a preset number of clusters.
Several different initialization methods are possible, but we use the Forgy method in which k random points from the data are selected as initial cluster centers.
One key weakness of the method is that the number of clusters must be fixed by the user initially. Inituitively, we can see that this data has 3 main clusters.
Each point is assigned to the cluster center which it is closest to (has the minimum Eucidian distance from) and then a new cluster centroid is found by finding the mean of all the points in a given cluster.
Each point is then reassigned to the center that is now closest to it.
The model has reached convergence once the centers of the clusters are no longer changing.
In this case, our model has correctly identified the 3 species into 3 separate clusters.
However, KMeans is vulnerable to its random initialization.
Here, runnign the same model with a different random seed gives us a different combination of clusters, on that less accurately reflects the three different species.
